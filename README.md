# Modelling Human Optimal Seeking Behaviour During Evaluation of Process Models With Subjective Complexity 

***Abstract***: This experiment presents an approach to modelling the optimal seeking behaviour of human experts during information perceiving using subjective complexity metrics understandability, interpretability and usability. A targeted survey captures expert’s perceived complexity evaluation during optimal selection of process models in a free-to-explore grid world space representing four real world healthcare procedures. We formulate state discovery rate, state visitaon entropy, and exploration score, represen ng optimal seeking behaviour. Using Markov Decision Processes (MDP) and Maximum entropy Inverse Reinforcement Learning (MaxEnt IRL), we model exploration–exploitation and topimal seeking behaviour of experts during information perceiving.
